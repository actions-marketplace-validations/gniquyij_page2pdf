# page2pdf

page2pdf saves web page to pdf using [Puppeteer](https://github.com/GoogleChrome/puppeteer).

## Usage
```
- uses: gniquyij/page2pdf@v1.0.0
  with:
    url: <url>   # E.g. 'https://gniquyij.github.io'
```
